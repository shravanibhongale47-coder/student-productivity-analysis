
# 📊 Student Productivity Analysis

## 📌 Description
This project analyzes student lifestyle and study habits to understand their impact on academic performance. Using SQL queries, the dataset was explored to identify key factors affecting GPA and productivity.

## 🛠 Tools & Technologies
- MySQL
- SQL (SELECT, JOIN, GROUP BY, AVG, COUNT)
- MySQL Workbench

## 📈 Key Features
- Data cleaning and preprocessing
- Analysis of study hours vs GPA
- Sleep pattern impact analysis
- Social media usage insights

## 📊 Results
- Identified strong correlation between study hours and GPA
- Observed negative impact of excessive social media usage
- Highlighted importance of balanced lifestyle

## 🚀 Conclusion
This project demonstrates how SQL can be used to extract meaningful insights from raw data and support decision-making.
